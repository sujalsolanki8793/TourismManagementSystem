# Tourism Management System

A comprehensive Java-based Tourism Management System developed using Core Java, JDBC, and MySQL. The application enables tourists to register, browse tour packages, make bookings, select hotels and transport services, process payments, manage bookings, and submit reviews. Administrators can efficiently manage tourists, tour packages, hotels, transport services, bookings, and payments through a dedicated admin panel.

### Key Features

* Tourist Registration & Login
* Tour Package Management
* Hotel & Transport Selection
* Secure Payment Processing
* Booking & Cancellation Management
* Customer Reviews & Ratings
* Booking History Export
* Admin Dashboard

### Technologies Used

* Java
* MySQL
* JDBC
* OOP Concepts
* Collections Framework
* File Handling
